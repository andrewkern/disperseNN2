Welcome to disperseNN2 documentation!
=====================================

**disperseNN2** is a deep learning framework for the analysis of
dispersal from population genetic data. See Smith et al. (2022)

Check out the :doc:`usage` section for further information, including how to :ref:`install` the project.

.. note::

   This project is under active development.
