Development
-----------

To install the current dev version:

.. code-block:: console

                (.venv) $ git clone https://github.com/chriscrsmith/disperseNN2.git
		(.venv) $ cd disperseNN2/
		(.venv) $ pip install . 

1. run ``black``
2. run ``flake8``, fix any issues
3. open a PR: https://github.com/kr-colab/disperseNN2
